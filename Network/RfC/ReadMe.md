# RfC you might  consider

## on Namespaces and special TLD

- [RfC2606 "Reserved Top Level DNS Names"](https://www.rfc-editor.org/rfc/rfc2606.html)
  - `.test`
  - `.example`
  - `.invalid`
  - `.localhost`
- [RfC6762 "Multicast DNS"](w.rfc-editor.org/rfc/rfc6762#appendix-G)
  
  Appendix G covers *private DNS namespaces* and suggest to use
  - `.intranet`
  - `.internal`
  - `.private`
  - `.corp`
  - `.home`
  - `.lan`
