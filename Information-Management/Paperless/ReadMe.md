Actually just collecting information ...

- [Paperless-ngx](https://github.com/paperless-ngx/paperless-ngx) is a document management system that transforms your physical documents into a searchable online archive so you can keep, well, *less paper*.

- [Paperless-AI](https://github.com/clusterzx/paperless-ai) is an AI-powered extension for Paperless-ngx that brings automatic document classification, smart tagging, and semantic search using OpenAI-compatible APIs and Ollama.

  It enables fully automated document workflows, contextual chat, and powerful customization — all via an intuitive web interface.
